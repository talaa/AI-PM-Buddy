# Company Processes

## BoQ Analysis Process
1. **Ingest BoQ** – Load the Excel/CSV file and normalize columns.
2. **Extract Relevant Items** – Filter rows for the requested material (e.g., ARDA).
3. **Aggregate Quantities** – Sum the `Total Qty` for the specified year.
4. **Validate Results** – Cross‑check totals against historical data.
5. **Generate Summary** – Produce a concise answer for the user.
